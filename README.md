# wikidata-reconciler-gs
A robust app to match spreadsheet data to Wikidata 

## Features
* Uses a Google Sheets sidebar.
* Fetches match candidates for a string in the current sheet line by line.
* Optionally also fetches labels and descriptions in any given language, including "mul", the multilingual language tag.
* Creates empty, named columns for the extended data.
* Provides links to Wikidata in the candidate list.
* Uses the Wikidata Action API.

Follow the repository for updates. The app will be available publicly soon.
